**Connection across this week:**
Ch. 20 poses the choice the whole book has been building toward: wealth/greatness, or wisdom/virtue? It closes asking what wisdom and virtue actually are. ## Common Pitfalls

| Pitfall | What students assume | What the text actually claims | How to avoid it |
|---|---|---|---|
| Treating vanity-enables-success as endorsement | Smith is praising self-admiration because it correlates with success | This is a critical, almost cynical observation, not a recommendation | |
| Missing why the third reason is "most important" | All three reasons for the virtue road carry equal weight | The littleness-based reason makes the *other* road actually unavailable, not just less appealing | |
| Treating "awful" as a value judgment | "Awful virtues" means bad or unpleasant virtues | Smith means "awe-inspiring" / commanding self-discipline, not morally negative | |
| Picking a side (tough vs. tender) | Smith ultimately privileges one virtue-set over the other | Perfect virtue requires both together — "you can't have one without the other" | |
| Treating "perfect virtue" as literally achieved | Some real person has actually achieved complete moral perfection | Smith explicitly denies absolute perfection is humanly attainable — "perfect virtue" talk means the best of the *relative* standard | |
| Concluding the absolute standard should be dropped | Since no one can reach absolute perfection, we should judge ourselves only by the common standard | Smith insists we keep the absolute standard in view anyway — losing it lets the bar for virtue drift to whatever's ordinary | |
