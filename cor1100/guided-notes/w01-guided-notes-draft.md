## Common Pitfalls

| Pitfall | What students assume | What the text actually claims | How to avoid it |
|---|---|---|---|
| Sympathy vs. impartial spectator | Same mechanism | Distinct: one feels-with, one judges as a neutral third party | |
| "Smith = capitalism" | Self-interest/market vocabulary applies here | This text is about moral psychology in TMS, not Wealth of Nations | |
| Self-interest = greed | Status- and luxury-seeking | Narrow drive toward basic self-care only | |
| Useful ≠ moral | Economically helpful = morally good | Two separate claims; only the first is automatic | |
| Interest in others = ordinary altruism | "Caring sometimes" | A stronger claim — others' happiness is "necessary" to us | |
| Resolving the division by picking a side | Consistently choosing self *or* others settles it | Requires ongoing balance — neither extreme is adequate | |

## Agent 4 — Reviewer Notes (not for students)

**All three improvements below were applied to this draft** (third-person scenario cues added to each Part; "locate the line" sub-prompts added to each Part's first cue; Matrices 1–2 merged). Kept for the record so the rationale stays attached to the template.

**Improvement 1 (Highest Impact): The template trains personal recall, not the quiz's actual task format**

Nearly every applied cue asks the student to generate their own life example: Part I cue 3 (Soren), Part III cue 3 (vicarious happiness), and all twelve rows of the Key Terms "Your example" column. The ch00/01/02 MCQ banks instead present third-person scenarios (Maya and Diego, the bakery owner, Fatima and Sara) and ask students to select which concept best fits. A student who only practices generating examples hasn't practiced evaluating someone else's situation against multiple candidate concepts — the actual transfer target.

*Fix:* Add one third-person scenario cue per Part (fresh entities, not bank scenarios) asking the student to identify which of two named concepts a brief situation illustrates, before the own-life application cue.

**Improvement 2: Application skips past the text itself**

Only Part III's opening-line cue asks students to locate or quote what Smith/Hanley actually wrote before applying it. Every other applied cue can be completed from memory of concepts alone, without returning to the text — the opposite of the close-reading habit a primary-text course should build.

*Fix:* In the first cue of each Part, add a sub-prompt: "Write or paraphrase, with a page reference, the specific line this section is built on" before the application question.

**Improvement 3: Four distinct table formats in one sitting is more reorientation than the content requires**

The document moves through a Key Terms table, three Cornell tables, four comparison matrices, and a Pitfalls table — seven format switches in one week. Matrices 1 and 2 (both simple binary contrasts from the Introduction) could merge into one four-row table, cutting a format switch with no loss of content.

*Fix:* Merge Matrices 1 and 2 into a single table (rows: the two virtue pairs and the two challenge types; columns: which side of human nature/the modern world, what it governs, your example).

**Overall Assessment:** The template's strongest feature is the throughline tracking sympathy/impartial spectator from the Introduction through their application in Part II (self-interest) and Part III (interest in others), rather than treating the week as three disconnected chapter summaries — the hardest design problem in a multi-chapter weekly document, solved correctly here. Preserve it through any further revision. The three gaps above concern calibrating practice tasks to the actual transfer target and trimming format-switching overhead, not the underlying conceptual architecture.
